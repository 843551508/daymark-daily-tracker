# 日迹使用教程

## 1. 第一次使用

打开页面后，右上角同步状态会显示以下两种情况：

- `已同步`：数据同时写入浏览器与 SQLite。
- `本地模式`：服务器不可用，数据仍会保存在当前浏览器。

顶部日期控制器决定当前正在查看和编辑哪一天。切换日期后，总览、健康、任务、日历详情会同步更新。

## 2. 今日总览

总览适合每天打开一次完成快速记录：

1. 选择 1 到 5 的心情评分。
2. 填写睡眠、饮水、步数和专注时长。
3. 在“今日一句”记录当天事件或复盘。
4. 点击“保存签到”。
5. 在右侧完成习惯打卡与今日任务。

完成度由签到字段、习惯和任务共同计算。当天没有任务时，任务不会拖低完成度。

## 3. 收支与预算

### 表单记账

进入“收支预算”，选择支出或收入，填写金额、日期、分类、账户和备注即可。

### 自然语言批量记账

在“批量快速记账”中可一次输入多笔：

```text
午饭35，打车18，工资+8000
```

系统会根据关键词推断收入或支出及分类。无法确定时归入“其他”。保存后仍可在明细表中编辑。

### 分类预算

1. 点击“本月预算”右上角的编辑按钮。
2. 为餐饮、交通、购物等分类填写月度额度。
3. 保存后即可查看已用比例与剩余额度。

月份选择器会同时影响月度统计、预算和趋势图。

## 4. 习惯打卡

“习惯打卡”页面提供完整周视图：

- 点击日期方格完成或取消打卡。
- 使用上一周、下一周查看历史。
- 新建习惯时可配置名称、每日目标、单位和颜色。
- 连续天数从当前选择日期向前计算。

删除习惯只删除习惯定义，历史打卡数据仍保留在备份中。

## 5. 健康与自定义指标

内置指标包括：

- 睡眠时长
- 体重
- 步数与饮水
- 运动和专注分钟
- 屏幕时间
- 精力与压力
- 感恩记录

点击“添加指标”可创建任意数字指标，例如阅读页数、热量、血压、用药次数或学习单词数。创建后会自动出现在每日健康表单和趋势选择器中。

## 6. 任务与目标

### 任务

任务支持日期和三档优先级，可按今天、待办、已完成、全部筛选。总览页只显示所选日期的任务。

### 目标

目标由名称、目标值、当前值、单位和截止日期组成。使用加减按钮更新进度，进度条会自动计算百分比。

## 7. 日历与洞察

统计日历会把以下数据聚合到每天：

- 收入与支出
- 习惯完成数
- 心情
- 任务完成数
- 当日日记

趋势洞察支持 7、30、90 天周期，展示完成度、平均睡眠、任务完成率、支出结构以及心情和睡眠变化。

## 8. 全局搜索

点击顶部搜索按钮可搜索：

- 收支备注、分类和账户
- 任务标题
- 目标名称
- 每日一句和感恩记录

点击搜索结果会切换到对应日期与模块。

## 9. 备份与恢复

### 完整备份

数据中心的“导出完整备份”会生成 JSON，包含所有模块、设置和历史记录。

### 导入备份

导入时采用合并策略：

- 同 ID 记录以导入文件为准。
- 不同 ID 记录全部保留。
- 日记录、预算和设置以导入文件覆盖同名键。

### CSV

- 收支页面导出当前月份的收支 CSV。
- 数据中心导出所有日期的汇总 CSV。

CSV 使用 UTF-8 BOM，可直接由 Excel 打开。

## 10. 推荐日常流程

1. 早晨添加任务并确认习惯。
2. 发生收支时快速记账。
3. 晚上填写签到和健康指标。
4. 每周查看习惯周表与趋势洞察。
5. 每月设置预算并导出一次完整备份。
